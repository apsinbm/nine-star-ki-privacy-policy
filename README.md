# Nine Star Ki Privacy Policy

This repository hosts the official Privacy Policy for the Nine Star Ki mobile application.

## Overview

The Nine Star Ki app is committed to privacy by design. This repository contains:

- **PRIVACY_POLICY.md** - Markdown version of the privacy policy
- **index.html** - Formatted HTML version for web viewing
- **README.md** - This file

## Key Points

- ✅ No personal data collection
- ✅ All calculations performed locally on device
- ✅ No analytics or tracking
- ✅ No third-party data sharing
- ✅ Fully compliant with GDPR, CCPA, and COPPA
